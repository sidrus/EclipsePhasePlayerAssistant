=== What is the Player Assistant?
This is a web application which is designed to keep track of your game-play data.  You can track campaign notes, inventory items, and credit at the moment.  The item database is customizable and you can easily add new items to your inventory.

=== What is Eclipse Phase?
Eclipse Phase is a pen and paper role-playing game set in a post-apocalyptic Sol-system.  For more information, visit: http://eclipsephase.com/
